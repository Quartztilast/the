# Лидерборд Яндекс Игр

В игре таблица лидеров не показывается внутри интерфейса. Код только отправляет лучший результат в Яндекс Игры через `ysdk.leaderboards.setScore()`.

## Что уже сделано в коде

- Техническое имя лидерборда: `catMergeLeaderboard`.
- Отправка результата происходит при обновлении рекорда.
- Для локального запуска используется fallback: если SDK недоступен, игра продолжает работать без ошибок.
- SDK подключается через `/sdk.js`, как требуется для архива Яндекс Игр.

## Как добавить лидерборд в консоли Яндекс Игр

1. Откройте черновик игры в консоли Яндекс Игр.
2. Перейдите во вкладку **Leaderboards**.
3. Создайте лидерборд.
4. В поле **Technical leaderboard name** укажите `catMergeLeaderboard`.
5. Тип значения выберите **score**.
6. Включите **Is main leaderboard?**, если хотите, чтобы Яндекс показывал рейтинг на карточке игры.
7. Сохраните черновик и проверьте игру в debug panel.

Если нужно другое техническое имя, поменяйте константу `LEADERBOARD_NAME` в `game.js` на такое же значение, как в консоли.
