# Котослияние

HTML5-игра для Яндекс Игр в жанре merge/suika: одинаковые котики соединяются в более крупных котиков, новый котик слегка взлетает после соединения, а результат сохраняется локально и отправляется в лидерборд Яндекс Игр.

## Запуск

```bash
npm run dev
```

Откройте `http://localhost:4173`.

## Проверки

```bash
npm run lint
npm run build
```

`npm run build` собирает минимальный архивный набор в `dist/` без исходных файлов с пробелами в именах.

## Яндекс Игры

- SDK подключается через `/sdk.js`.
- `LoadingAPI.ready()` вызывается после готовности интерфейса.
- `GameplayAPI.start()`/`stop()` используются при старте, паузе, рекламе и завершении.
- Лидерборд не отображается в игре, но рекорд отправляется в технический лидерборд `catMergeLeaderboard`.
- Инструкция по созданию лидерборда: `docs/yandex-leaderboard.md`.
